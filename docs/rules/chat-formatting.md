# Разметка чата
Для обозначения различной информации на игровом сервере, в текстовом чате, используется приведённая разметка

## Текстовый чат

Разметки нет - RP сообщение

\*`Действие`\* - RP действие

\*Т\* `Текст` - Звонок по телефону

\*Р\* `Текст` - Сообщение по рации

<`Текст`> - Шёпот

((`Текст` - Non-RP сообщение

## Голосовой чат

Разметки нет - RP информация

Все RP действия писать в текстовый чат

"По Нон РП / Нон РП информация `Информация`" - Non-RP информация

"Шёпот `Информация`" - Шёпот
